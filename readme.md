# TravelWorld Blog

A modern, responsive travel blog website built with HTML, CSS, and JavaScript. TravelWorld Blog features destination guides, travel tips, and stunning photography from around the world.

## Live Demo

Check out the live website: https://travelworldblog.netlify.app/

## Features

- Responsive design that works on mobile, tablet, and desktop
- Clean, modern user interface with travel-themed styling
- Featured destinations section
- Travel tips and guides blog
- Newsletter subscription form
- Social media integration
- Fast loading and optimized performance

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, and custom properties)
- JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts
- Netlify (for hosting)

## Setup Instructions
1. Clone the repository:
    git clone https://github.com/shayansk10/travelworldblog.git

2. Navigate to the project directory:
    cd travelworldblog

3. Open `index.html` in your browser or use a local server.

## Customization

To customize this travel blog for your own use:

1. Update the website title and metadata in the `index.html` file
2. Replace the images in the `images/` folder with your own travel photos
3. Modify the color scheme by changing the CSS custom properties in `style.css`
4. Update the content with your own travel experiences and tips
5. Change the contact information and social media links in the footer

## Deployment

The site is deployed on Netlify. To deploy your own version:

1. Fork this repository
2. Connect your GitHub account to Netlify
3. Select this repository for deployment
4. Netlify will automatically deploy your site
